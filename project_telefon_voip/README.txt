Potrebné knižnice a programy:
- python3
- Django 3.0 pre python (inštalovateľné cez pip)
- Pillow pre Django (inštalovateľné cez pip)

Projekt sa spúšťa z konzoly:

Windows:
python manage.py runserver

Linux: (netestované)
python3 manage.py runserver

=> následne v prehliadači ísť na localhost:8000 (127.0.0.1).

Okomentované sú súbory v template/voip a v static/voip (js.js)...
